Character Encoding Translator

Character Encoding Translator is a Java GUI application, console application, and API for converting text files from one character encoding to another.

Download this project as a .zip file Download this project as a tar.gz file

Dealing with unmappable characters

Character Encoding Translator supports the ability to replace unmappable characters with their numeric character reference equivalents ("&#NNNN;"), a useful feature when preparing text for use on the Web.
